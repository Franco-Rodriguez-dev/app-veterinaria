# 🐾 App Veterinaria

Aplicación fullstack para la gestión de mascotas, desarrollada con arquitectura separada (frontend + backend).

## 🚀 Tecnologías utilizadas

### Frontend

* Angular
* TypeScript
* RxJS
* HTML / CSS

### Backend

* .NET (ASP.NET Core)
* C#
* API REST

---

## 🔐 Funcionalidades principales

* Autenticación de usuarios (login)
* Gestión de mascotas (CRUD)
* Manejo de roles
* Persistencia de sesión con token
* Comunicación frontend-backend mediante API REST

---

## 🧱 Arquitectura

El proyecto está dividido en dos partes:

* `FE-CRUDMascotas` → Aplicación Angular (cliente)
* `BE-CRUDMascotas` → API REST en .NET (servidor)

---

## ▶️ Cómo ejecutar el proyecto

### 🔹 Backend (.NET)

```bash
cd BE-CRUDMascotas
dotnet run
```

---

### 🔹 Frontend (Angular)

```bash
cd FE-CRUDMascotas
npm install
ng serve
```
